# Python Contact Manager

## Project Description
This is a simple command-line Contact Manager application built using Python.  
It allows users to add, view, and delete contacts from a contact list.

## Features
- Add new contacts
- View saved contacts
- Delete contacts
- Simple command-line interface

## Technologies Used
- Python
- Git
- GitHub

## How to Run the Application

1. Clone the repository

git clone https://github.com/your-username/Python-app.git

2. Navigate to the project folder

cd Python-app

3. Run the application

python Contacts.py
